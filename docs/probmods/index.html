Probabilistic Models of Cognition

An interactive textbook on probabilistic programming and Bayesian cognition, using prob-cljs (ClojureScript) instead of Church.

This is a port of chapters 1–9 of Probabilistic Models of Cognition v1 by Noah Goodman and Joshua Tenenbaum. All code examples are editable and runnable directly in the browser via Scittle.

Chapters

  1. Introduction
  2. Generative Models
  3. Conditioning
  4. Patterns of Inference
  5. Models for Sequences of Observations
  6. Inference about Inference
  7. Algorithms for Inference
  8. Learning as Conditional Inference
  9. Hierarchical Models
