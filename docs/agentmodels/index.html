Modeling Agents with Probabilistic Programs

An interactive textbook on planning, decision-making, and inference about agents, using prob-cljs (ClojureScript) instead of WebPPL.

This is a port of agentmodels.org by Owain Evans, Andreas Stuhlmüller, John Salvatier, and Daniel Filan. All code examples are editable and runnable directly in the browser via Scittle.
